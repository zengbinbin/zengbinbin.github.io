mac下安装nvm---node版本管理工具

发表于 2018-03-26

前言

因为项目的原因只能安装8.x的node，倘若不是安装8.x的就会报错,因此便想到了使用nvm来管理Node

一.安装brew

终端上运行

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

二。安装指定版本的node

  • 需要安装homebrew-version 安装nvm
brew install nvm
  • 查看可安装的node版本
source $(brew --prefix nvm)/nvm.sh                 //唤醒nvm


nvm ls-remote查看所有node 版本


nvm install version 就可以安装你想要的版本 .    ----
  • 切换node的版本
nvm use --delete-prefix version
  • 查看已安装版本
nvm ls  //查看已经安装的版本


注：绿色箭头的为已经在使用版本
